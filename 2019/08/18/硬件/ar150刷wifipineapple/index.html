准备

下载AR150路由的wifipineapple的rom，AR150-wifipine固件下载
备份现有路由器版本的rom

刷机

有两种方式：

  1. 利用路由器升级功能
  2. 把rom上传到路由器，然后执行刷机命名，等待完成
    mtd -r write /tmp/xxxx.rom firmware

    Ps：中间不要断电

刷机完成之后，浏览器地址输入：172.16.42.1:1471，进入设置界面

以下是图解，这里采用的是路由器升级功能




如果你卡在了，启动阶段
尝试ssh连接后，输入以下命令，等待重启完成

jffs2reset -y && reboot
